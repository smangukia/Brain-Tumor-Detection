About This Project

The Technology

This brain tumor detection system uses a deep learning model based on MobileNet architecture with transfer learning. The model was trained on a dataset of MRI scans and achieves over 97% accuracy in detecting the presence of brain tumors.

How It Works

When you upload an MRI scan image, the system:

  1. Preprocesses the image to match the format expected by the model
  2. Passes the processed image through the neural network
  3. Analyzes the output to determine the probability of a tumor
  4. Presents the results with a confidence score

Model Architecture

The model uses a pre-trained MobileNet as the base, with the following modifications:

  • Frozen base layers to preserve learned features
  • Custom classification head for binary tumor detection
  • Trained with specialized medical imaging preprocessing

Limitations

While our model performs well, it has several limitations:

  • It is not a substitute for professional medical diagnosis
  • Performance may vary based on image quality and scan type
  • The model cannot determine tumor type, size, or location
  • It should be used as a preliminary screening tool only
